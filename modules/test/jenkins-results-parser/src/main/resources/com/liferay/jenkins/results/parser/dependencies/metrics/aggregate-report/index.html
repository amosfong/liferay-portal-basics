Category Descriptions

liferay-portal/master PR's: This accounts for all CI builds (all test suites) related to the liferay-portal/master pull request jobs. For detailed information, please see the Pull Request History Report.

liferay-portal/master Upstream: This accounts for all CI builds related to liferay-portal/master upstream jobs. This includes acceptance upstream and test suite upstream jobs. For detailed information, please see the Upstream History Report.

Portal Release: This accounts for all CI builds related to fixpack, hotfix, and release jobs. This ncludes master and legacy branches. For detailed information, please see the Release History Report.

liferay-portal-ee PR's & Upstream: This accounts for all CI builds for pull request and upstream jobs related to non master branches.

CI Maintenance: This accounts for CI server upkeep jobs such as verification, maintenance daily, and maintenance weekly.

Other: This accounts for all other job types not in an above category.
